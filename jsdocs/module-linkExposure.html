Module: linkExposure

This module enables measuring user exposure to linked content. See the onLinkExposureData and onLinkExposureUpdate events for specifics. There is an important difference between these events: onLinkExposureData fires once per page with a complete set of link exposure data, while onLinkExposureUpdate fires throughout a page's lifespan as link exposures occur. For most use cases, onLinkExposureData is the right event to use.
